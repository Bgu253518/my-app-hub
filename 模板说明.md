# 🎬 AI 视频讲解生成模板

> 基于 Remotion + Edge-TTS 的自动化 AI 讲解视频生成方案

---

## 一、项目结构

```
项目1 视频讲解/
├── package.json           # 项目配置 & npm 脚本
├── tsconfig.json          # TypeScript 配置
├── remotion.config.ts     # Remotion 渲染配置 (含Chrome路径)
├── setup.bat              # 首次运行: 安装所有环境依赖
├── render.bat             # 一键渲染: TTS + 视频
├── generate_audio.py      # TTS 语音生成脚本
├── 模板说明.md             # 本文件
├── src/
│   ├── index.ts           # 入口, 注册根组件
│   ├── Root.tsx           # 视频编排 (场景/时间线)
│   └── components/
│       ├── ParticleBackground.tsx  # 粒子背景动画
│       ├── TypewriterText.tsx      # 打字机效果
│       └── AnimatedNumber.tsx      # 数字滚动动画
├── public/
│   └── audio/
│       └── narration.mp3  # [自动生成] TTS 旁白
└── out/
    └── video.mp4          # [渲染输出] 最终视频
```

## 二、快速开始

### 首次使用
```bash
# 双击运行 setup.bat
# 或手动执行:
npm install
pip install edge-tts
# setup.bat 会自动从 npmmirror 镜像下载 Chrome Headless Shell
```

### 每次使用
```bash
# 双击 render.bat (一键: TTS + 渲染)
# 或分步执行:
python generate_audio.py
npm run render
```

> **注意**: Remotion 需要 Chrome Headless Shell 来渲染视频。
> 如果国内下载失败, setup.bat 会自动使用 npmmirror.com 镜像下载 (约113MB)。

## 四、自定义指南

### 修改视频时长

编辑 `src/Root.tsx`:
```tsx
// 修改总帧数 (durationInFrames)
<Composition
  id="AiVideo"
  component={MainVideo}
  durationInFrames={600}   // 20秒 = 600帧
  fps={30}
  width={1920}
  height={1080}
/>
```

同时调整各场景的 `START` 和 `DURATION`:
```tsx
const SCENE_1_START = 0;
const SCENE_1_DURATION = 150;  // 加大
const SCENE_2_START = 150;
const SCENE_2_DURATION = 200;
// ... 以此类推
```

### 添加新场景

在 `Root.tsx` 中:
1. 定义 `SCENE_5_START` 和 `SCENE_5_DURATION`
2. 写一个 `Scene5` 组件 (参考现有场景)
3. 在 `MainVideo` 中添加 `<Sequence>`

### 修改配色

编辑 `ParticleBackground.tsx` 中的颜色值:
```tsx
// 粒子颜色
colors: ["rgba(0, 229, 255, OPACITY)", ...]

// 背景渐变
background: "linear-gradient(135deg, #050510 0%, ...)"

// 光晕效果
background: "radial-gradient(ellipse at center, rgba(0,229,255,0.08)...)"
```

### 修改动画参数

所有弹簧动画使用 `spring()`:
```tsx
spring({
  frame,
  fps: 30,
  config: {
    damping: 10,    // 阻尼: 越大越快停 (5-20)
    stiffness: 80,  // 刚度: 越大越弹 (50-200)
  },
  durationInFrames: 40,
});
```

### 替换背景

将 `ParticleBackground` 替换为:
- 图片: 使用 Remotion 的 `<Img>` 组件
- 视频: 使用 `<OffthreadVideo>` 组件
- 纯色: 直接用 `<div>` + 背景色

## 五、可用动画组件

### TypewriterText — 打字机效果
```tsx
<TypewriterText
  text="你的文字"
  startFrame={0}          // 从第几帧开始
  charDuration={4}        // 每个字出现间隔 (帧数)
  glowColor="#00e5ff"     // 发光颜色
/>
```

### AnimatedNumber — 数字滚动
```tsx
<AnimatedNumber
  value={100}             // 目标数字
  startFrame={10}         // 开始帧
  durationFrames={45}     // 动画持续帧数
  prefix="增长 "          // 前缀文字
  suffix=" %"             // 后缀文字
  glowColor="#b44dff"     // 发光颜色
/>
```

## 六、TTS 旁白最佳实践

### 语速控制
在 `generate_audio.py` 中:
```python
communicate = edge_tts.Communicate(TEXT, VOICE, rate="+5%")
# rate: "+20%" 加快, "-10%" 减慢, "+0%" 正常
```

### 文案时长估算
- 中文正常语速: ~4字/秒
- 10秒视频约 35-45 个汉字
- 先录音, 再根据实际时长调整场景帧数

### 多段旁白
改为逐句生成, 在 Remotion 中按帧放置:
```python
texts = [
    "第一句话...",
    "第二句话...",
]
# 生成多个 mp3, 在 Remotion 中用多个 <Audio> 按帧偏移放置
```

## 八、常见问题

**Q: 渲染时报 chrome 错误**
A: Remotion 需要 Chrome/Chromium。运行 `npx remotion browser` 确保浏览器可用。

**Q: 中文显示乱码**
A: 检查系统是否安装了中文字体 (微软雅黑/PingFang SC)。可在代码中调整 `fontFamily`。

**Q: 音频不同步**
A: 先用 `ffprobe` 检查音频时长, 再调整场景帧数匹配时长。

**Q: 想要更长的视频**
A: 增加 `durationInFrames`, 扩展各场景时长, 增加场景数量, 延长 TTS 文案。

## 九、进阶: 全自动化流水线

可以编排为完全无人值守的生成流程:

```
1. 输入文案/主题
      ↓
2. AI 生成分镜脚本
      ↓
3. 脚本生成 TTS 旁白
      ↓
4. 脚本驱动 Remotion 场景
      ↓
5. 自动渲染 → 输出视频
```

Claude Code 可以协助完成步骤 1-4 的代码编写。
