一、工具箱简介

审计工具箱是一套专为审计、财务人员打造的纯浏览器端工具集合。所有工具在用户本地浏览器中运行，无需安装任何软件，不依赖后端服务器，数据永不上传。工具箱涵盖 PDF 处理、文件重命名、数据清洗、图片压缩、租赁测算、Word 报告校验、底稿滚存、TB 上数、审计抽凭、Claude Code 配置等多个实用工具，所有功能免费使用。

技术特点：纯前端运行 · 本地数据安全 · 零安装 · 深色玻璃态设计 · 响应式布局支持移动端。

二、数据安全与隐私声明

2.1 数据本地化

所有工具均在用户浏览器中运行，文件数据不会上传至任何远程服务器。PDF 处理、图片压缩、CSV 清洗等涉及文件的操作，全部在本地浏览器（WebAssembly / Canvas / File API）中完成。网络请求仅用于加载页面资源（CDN 脚本库），不包含用户数据。

2.2 隐私保障

本工具箱不收集用户信息，不使用 Cookie 追踪，不嵌入第三方分析脚本。AI 助手模型下载完成后支持完全断网使用。密码锁功能使用浏览器 sessionStorage，密码仅在本地比对，不上传。

2.3 安全建议

  • 处理敏感文件时，建议在可信网络环境中使用
  • 使用完毕后关闭浏览器标签页以清除 sessionStorage
  • 不要在公共电脑上勾选"记住密码"

四、通用操作说明

4.1 文件上传

支持文件拖拽上传和点击选择文件两种方式。拖拽区域有视觉反馈（边框高亮），PDF 文件上传后右侧面板自动预览前几页。

4.2 参数设置

各工具提供相应的参数选项（下拉框、滑动条、文本输入等），参数修改实时保存，点击"执行"或"开始"按钮启动处理。

4.3 结果导出

处理完成后，显示"下载结果"按钮或自动触发浏览器下载。部分工具支持导出为 CSV、Excel、ZIP 等格式。

4.4 进度反馈

大型文件处理时，页面显示处理中动画（旋转加载器），避免重复点击。PDF 渲染使用 pdf.js Worker 在后台线程执行。

五、常见问题与注意事项

5.1 浏览器兼容性

  • 推荐使用 Chrome / Edge 浏览器（最新版本）
  • Firefox / Safari 部分功能（如 File System Access API）暂不支持
  • 移动端浏览器可能因性能限制处理较慢

5.2 文件大小限制

  • PDF 文件建议不超过 100MB（浏览器内存限制）
  • 图片批量处理建议单次不超过 50 张
  • CSV 数据行数建议不超过 10 万行

5.3 性能优化建议

  • 处理大文件时关闭其他占用内存的标签页
  • 图片压缩质量建议 60-80%（兼顾体积与质量）
  • PDF 拆分图片时选择 2x 分辨率、PNG 格式获得最佳清晰度

5.4 常见问题

Q：PDF 页面预览不显示？
A：检查浏览器是否支持 Web Worker，确认 pdf.js 加载成功。刷新页面后重试。

Q：文件重命名后如何撤销？
A：重命名执行后点击"撤销"按钮（保持页面不刷新），可恢复原始文件名。

Q：AI 助手加载失败？
A：首次使用需下载模型（约 600MB），请等待下载完成。如长时间卡住，刷新页面重试。推荐使用 Chrome（支持 WebGPU 加速）。

Q：忘记工具箱密码？
A：密码为预设值，请联系工具箱管理员获取。清除浏览器 sessionStorage 后需重新输入。
