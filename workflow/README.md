# 创作者智能片场 v3.3.0

5车间18工位自动化视频生产工作流：创意熔炼 → 叙事锻造 → 视觉翻译 → 自动装配 → 分发进化

## 架构概览

```
车间0：创意熔炼         车间1：叙事锻造         车间2：视觉翻译         车间3：自动装配         车间4：分发进化
┌─────────────────┐   ┌─────────────────┐   ┌─────────────────┐   ┌─────────────────┐   ┌─────────────────┐
│ 0.1 核心观点打磨  │ → │ 1.1 叙事框架选择  │ → │ 2.1 全片视觉定调  │ → │ 3.1 音频轨道生成  │ → │ 4.1 多平台文案    │
│ 0.2 素材弹药装填  │   │ 1.1b 视频时长选择 │   │ 2.2 分镜脚本生成  │   │ 3.2 视频合成渲染  │   │ 4.2 资产归档学习  │
│ 0.3 多维角度爆破  │   │ 1.2 口播文案锻造  │   │ 2.3 分镜审核      │   │ 3.3 终检抽检      │   └─────────────────┘
│ 0.4 角度满意？    │   │ 1.3 朗读审核      │   └─────────────────┘   └─────────────────┘         ↓
└─────────────────┘   │ 1.4 文案通过？    │                                                 End
                      └─────────────────┘
```

## 核心公式

**文案字数 = 视频时长(秒) × 4~5字/秒**

| 时长 | 字数 | 镜头数 |
|:----:|:----:|:------:|
| 15秒 | 60~75字 | 3~4个 |
| 30秒 | 120~150字 | 5~7个 |
| 60秒 | 240~300字 | 8~12个 |
| 90秒 | 360~450字 | 12~16个 |

## 防御性设计

- 3层时长匹配校验（源头约束 → 前置压缩 → 审核拦截）
- 4个回退分支（每车间有重来机制）
- 所有指令含异常处理（输入缺失/素材不足/字数超标）
- 满10期自动生成效能简报，驱动工作流迭代

## 技术栈

- edge-tts（中文语音合成）
- render_engine.js + puppeteer（视频渲染）
- ffmpeg（视频编码）
- CC Workflow Studio（工作流编排）
