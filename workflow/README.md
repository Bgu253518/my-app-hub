## v4.0.0 核心升级：Visual Director Engine

### 情感节拍分析（起承转合）
| 幕 | 功能 | 情感基调 | 时长占比 | 视觉策略 |
|:--:|------|---------|:--------:|---------|
| 起 | 吸引注意，抛出钩子 | 好奇/冲击 | ~20% | 大字冲击/极简构图 |
| 承 | 建立逻辑，给出论据 | 理性/信任 | ~35% | 数据可视化/线性叙事 |
| 转 | 制造冲突，提出挑战 | 紧张/反思 | ~30% | 对比构图/色彩转折 |
| 合 | 升华总结，引导行动 | 温暖/力量 | ~15% | 情感画面/留白收束 |

### 12种动画 - 语义映射
- **抛出观点** → elasticOut + scaleBounce
- **数据展示** → countUp
- **强调/脉动** → pulse + glow
- **对比/转折** → splitSlide + erase
- **情绪渲染** → blurIn
- **高潮汇聚** → particleGather
- **概念连接** → flowLine
- **冲突/警示** → glitch (text_effect)
- **列表/步骤** → slideIn
- **图表生长** → growUp
- **细微呈现** → fadeIn + scale
- **色彩过渡** → colorShift

### 5种转场 - 逻辑词映射
| 逻辑词 | 转场 | 时长 |
|-------|------|:----:|
| 比如/例如 | pushIn + crossFade | 0.4s |
| 但是/然而 | glitchTransition | 0.3s |
| 对比/另一方面 | flipCut | 0.5s |
| 此外/还有 | swipeLeft | 0.3s |
| 所以/因此 | fadeToBlack | 0.6s |

### 8种音效标记
ticktock / boom / heartbeat / piano / whoosh / digital / camera / bass

### 4种后期特效
chromatic(色差) / vignette(暗角) / glow(发光) / grain(噪点)

### 新增第4视觉风格：极简人文+科技冷感
#0a0a0a 背景 / #e8e8e8 文字 / 噪点纹理 / 微发光 / 90%以上模型禁用渐变色块和 emoji

## 核心公式

**文案字数 = 视频时长(秒) × 4~5字/秒**

| 时长 | 字数 | 镜头数 |
|:----:|:----:|:------:|
| 15秒 | 60~75字 | 3~4个 |
| 30秒 | 120~150字 | 5~7个 |
| 60秒 | 240~300字 | 8~12个 |
| 90秒 | 360~450字 | 12~16个 |

## 防御性设计

- 3层时长匹配校验（源头约束 → 前置压缩 → 审核拦截）
- 4个回退分支（每车间有重来机制）
- 所有指令含异常处理（输入缺失/素材不足/字数超标）
- 情感四幕完整性校验（起承转合缺一不可）
- 动画-语义匹配度审核（动画必须服务于信息传递）
- 视觉隐喻贯穿性检查（至少2场景）
- 满10期自动生成效能简报

## 技术栈

- edge-tts（中文语音合成）
- **Remotion 4.0 + VisualDirector.tsx（视频渲染引擎）**
- @remotion/transitions（内置转场库）
- ffmpeg（视频编码）
- CC Workflow Studio（工作流编排）
