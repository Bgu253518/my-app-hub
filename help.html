📝 批量文件重命名工具 已上线

纯浏览器端批量文件重命名，无需安装任何软件。三种重命名模式，支持预览和撤销。

操作步骤

  1. 点击上传区域选择文件夹（需 Chrome/Edge 浏览器）
  2. 文件列表自动加载，可按扩展名过滤
  3. 选择重命名模式： 查找替换（支持正则）、序号前缀（可排序）、自定义模板（{name}{num}{ext}{date}{parent} 等变量）
  4. 点击"预览效果"查看结果
  5. 确认无误后"执行重命名"，支持一键撤销

自定义模板示例

模板：{date}_Report_{num:03}{ext} → 结果：2026-05-13_Report_001.pdf

🧹 CSV 数据清洗器 已上线

上传表格自动扫描数据问题，一键清洗，专为 Power BI 导入场景优化。

操作步骤

  1. 上传 CSV / XLS / XLSX 文件

  5. 导出清洗后 CSV 或带报告的 Excel

📊 ROU 租赁测算器 已上线

基于 CAS 21（IFRS 16）的使用权资产标准化测算工具，自动生成摊销明细表。

操作步骤

  1. 录入租赁合同参数：付款金额、频率（月/季/半年/年）、折现率、租赁期
  2. 选择付款方式：期初/期末付款
  3. 可选参数：复原成本、租赁激励、余值担保
  4. 自动计算：租赁负债现值、ROU 资产原值
  5. 生成逐期摊销明细表（利息+本金+折旧+ROU净值）
  6. 含标准会计分录参考
  7. 支持 Excel 批量导入导出

⚙️ Claude Code 个人配置 已上线

个人加密金库，安全存储 Anthropic / OpenAI API 密钥。AES-256-GCM 加密，PBKDF2 密钥派生，所有数据仅存储于浏览器 localStorage。

安全机制

  1. 固定密码解锁（只有你知道）
  2. API 密钥经 AES-256-GCM 加密后存 localStorage
  3. 加密密钥通过 PBKDF2 派生（200,000 次 SHA-256 迭代）
  4. 此页面加载后零网络请求，无统计数据上报、无云端同步
  5. 换电脑：Export Encrypted Backup → USB → 新电脑 Import
安全保证：你的 API 密钥明文从未出现在任何服务器上。源码中仅存密码的 SHA-256 哈希（不可逆）。所有数据物理上只在你当前浏览器的 localStorage 磁盘文件中。

一键搭建脚本

Claude Vault 页面底部提供 ClaudeCodeSetup.bat 下载。任何新 Windows 电脑双击运行：自动下载便携 Node.js + Git + Claude Code CLI，打开金库页面输入密码和 API Key 即可使用。
