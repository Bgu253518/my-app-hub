Claude Code 一键配置工具

快速配置第三方 API 服务商，启用 Claude Code

接口地址：https://api.deepseek.com
默认模型：deepseek-chat
点击预览脚本内容 
请填写 API 密钥后点击"生成配置脚本"预览
使用步骤：
  1. 选择服务商，填写 API 密钥
  2. 点击「生成配置脚本」，预览确认无误
  3. 点击「下载 setup.bat」保存到本地
  4. 双击 setup.bat 即可自动配置（无需管理员权限）
  5. 如果自动配置失败，可从网页下载 claude-start.bat 备用包装器（无需注册表）
  6. 关闭当前终端，重新打开终端后运行 claude 即可
