1. PDF 多功能工具箱

纯浏览器端 PDF 综合处理 — 合并、拆分、提取文字、查看信息、加解密，一站式解决
🔐 隐私安全所有文件在您的浏览器本地处理，不会上传到任何服务器。处理大型 PDF 也不消耗流量。

功能一览

功能说明输入输出
📎 合并 PDF将多个 PDF 文件按顺序合并为一个2+ 个 PDF 文件1 个合并后的 .pdf
✂️ 拆分 PDF按页码范围提取指定部分为独立文件1 个 PDF + 页码范围1 个拆分后的 .pdf
📝 提取文字提取 PDF 中所有可复制文字内容1 个 PDF 文件.txt 文本文件
ℹ️ 文件信息查看元数据（页数、作者、创建时间等）1 个 PDF 文件屏幕展示
🔒 加密保护为 PDF 添加 AES-256 密码保护1 个 PDF + 密码1 个加密 .pdf
🔓 解密移除移除 PDF 的打开密码（需知道密码）1 个加密 PDF + 密码1 个解密 .pdf

操作步骤

  1. 选择功能

    在顶部标签栏点击对应功能：合并 / 拆分 / 提取文字 / 信息 / 加密 / 解密。默认进入「合并」。

  2. 上传文件

    将 PDF 文件拖拽到「拖拽 PDF 文件到此处」区域，或点击该区域选择文件。支持多选（合并模式下）。上传后右侧会预览 PDF 前几页。

  3. 配置参数（按需）

    拆分模式：输入页码范围（如 1-5,8,10-12）。加密模式：输入密码（至少 4 位）。解密模式：输入原密码。

  4. 执行处理

    点击底部操作按钮（如「合并并下载」）。处理过程中按钮会显示进度状态。

  5. 下载结果

    处理完成后自动触发浏览器下载。检查下载文件夹中的输出文件。

💡 提示拆分页码范围支持多种格式：单页 5、范围 3-8、多组 1-3,7,10-15。页码从 1 开始计数。
⚠ 注意加密采用 AES-256 标准。密码一旦丢失无法找回，请妥善保管。解密需提供原密码。

2. 图片批量压缩工具

批量压缩 JPG/PNG/HEIC 图片，合并转 PDF，HEIC 格式转换，纯浏览器端处理

功能一览

功能说明
🗜️ 批量压缩对多张图片同时压缩，可调质量（10%-100%），预览压缩前后大小对比
📄 转 PDF将多张图片合并为一个 PDF 文件，可调页面尺寸和方向
📱 HEIC 转换将 iPhone 拍摄的 HEIC 格式自动转换为通用 JPG/PNG 格式

操作步骤 — 批量压缩

  1. 上传图片

    拖拽图片文件（JPG/PNG/WebP/HEIC）到上传区域，或点击选择。支持一次选择多张。

  2. 调节压缩质量

    使用滑动条调节质量（10%-100%）。越低压缩率越高、文件越小、画质越低。右侧实时显示预估大小。

  3. （可选）调整尺寸

    可设置最大宽度/高度（像素），超出会等比缩放。不设为保持原尺寸。

  4. 预览对比

    每张图片下方显示「原始大小 → 压缩后预估大小」的对比。

  5. 下载

    点击「批量下载」打包为 ZIP 一次性下载所有压缩后的图片。

操作步骤 — 图片转 PDF

  1. 切换到「图片转PDF」标签
  2. 上传图片

    拖拽需要合并的图片。可拖拽调整图片顺序（排序影响 PDF 中的页面顺序）。

  3. 选择页面设置

    纸张尺寸（A4/Letter/原始）、方向（纵向/横向）、边距。

  4. 生成并下载

    点击「生成 PDF」，自动下载合并后的 PDF 文件。

💡 技巧建议质量设为 70-85%，文件大小可减少 60-80% 而肉眼几乎看不出差异。HEIC 格式会自动识别并转换。

3. CSV 数据清洗器

上传表格自动扫描问题，一键清洗：去空格、去重、标准化日期/数字、填充空值，导出 Power BI 就绪数据

支持的文件格式

CSV（逗号/制表符分隔）、XLS、XLSX。自动检测编码（UTF-8、GBK、BIG5 等）。

清洗功能详解

清洗项作用示例
去除首尾空格删除单元格内容前后的空白字符" 张三 " → "张三"
删除空行移除所有单元格都为空的行—
去重按所有列检测并删除完全重复的行保留首次出现的行
标准化日期将各种日期格式统一为 YYYY-MM-DD2024.12.31 → 2024-12-31
标准化数字去除千位分隔符、统一小数点、处理百分比¥1,234.56 → 1234.56
清理不可见字符去除零宽空格、制表符、换行符等—
填充空值将空单元格填充为指定值（0/上下行/自定义）空 → 0 或用上行值填充
删除全空列移除整列都无数据的列—

操作步骤

  1. 上传文件

    拖拽 CSV/Excel 文件到上传区域。系统自动扫描并显示数据预览和问题统计（空值数、重复行数、日期格式问题数等）。

  2. 勾选清洗项

    默认已全选推荐的清洗项。可根据需要取消勾选不需要的操作。

  3. （可选）配置填充策略

    如果选择「填充空值」，可选择填充方式：填充为 0、用上一行值填充、用下一行值填充、填充自定义值。

  4. 预览对比

    切换「原始数据 / 清洗后数据」标签查看清洗前后对比。问题行会用颜色高亮标记。

  5. 执行清洗

    点击「一键清洗」按钮。系统按勾选的清洗项依次处理，完成后显示清洗报告。

  6. 导出结果

    点击「导出 CSV」下载清洗后的数据。也可选择「导出报告 Excel」获得包含清洗前后对比 + 清洗日志的完整 Excel 文件。

⚠ 注意清洗操作不可撤销。建议先导出原始数据备份，或使用「导出报告 Excel」保留清洗前的数据对照。

4. 批量文件重命名工具

浏览器中一键批量重命名文件：查找替换、序号前缀、自定义模板三种模式，支持预览和撤销
🌐 浏览器要求需使用 Chrome 或 Edge 浏览器（需要 File System Access API）。Firefox 和 Safari 暂不支持。

三种重命名模式

模式适用场景示例
🔍 查找替换批量替换文件名中的特定文字将 "草稿_" 替换为 "终版_"；支持正则表达式
🔢 序号前缀添加统一前缀 + 自动编号报告_001.pdf, 报告_002.pdf...
📐 自定义模板用变量自由组合文件名{num}-{name}-{date}.{ext}

模板变量说明

变量含义示例
{name}原文件名（不含扩展名）report
{ext}原扩展名pdf
{num}自动序号（从起始值递增）001, 002, 003...
{date}当前日期2026-05-17
{parent}所在文件夹名称审计底稿

操作步骤

  1. 选择文件夹

    点击「选择文件夹」按钮，浏览器会弹出文件夹选择对话框。选择一个文件夹后，系统列出其中所有文件。

  2. （可选）过滤文件类型

    在扩展名过滤框中输入要处理的文件类型（如 .pdf,.xlsx），只显示匹配的文件。

  3. 选择重命名模式

    切换标签选择模式：查找替换 / 序号前缀 / 自定义模板。

  4. 配置规则

    按模式填写参数。查找替换可勾选「正则表达式」进行高级匹配。序号模式可设置起始编号和位数。

  5. 预览结果

    右侧表格会实时显示「原文件名 → 新文件名」的映射。检查是否有冲突（红色标记）。

  6. 执行重命名

    点击「执行重命名」。完成后可点击「撤销」恢复到原名。

⚠ 注意操作的是本地文件系统中的真实文件。重命名后关闭页面即无法撤销，建议先预览确认无误再执行。

5. ROU 租赁测算器

基于 CAS 21（IFRS 16）新租赁准则的标准化测算工具 — 多合同批量测算，自动生成完整摊销明细表

支持的租赁参数

参数说明典型值
租赁年限租赁合同总时长3年 / 5年 / 10年
付款频率每年付款次数月付(12) / 季付(4) / 半年付(2) / 年付(1)
付款时点期初付款或期末付款期初 / 期末
折现率计算租赁负债现值的利率4.5% - 8%（增量借款利率）
每期付款额定期支付的租金金额如月付 ¥50,000
复原成本租赁期满预计的复原费用可选，如 ¥100,000
租赁激励出租方给予的优惠可选，如免租期
余值担保承租方担保的残值可选

操作步骤

  1. 添加合同

    点击「添加合同」，输入合同名称（如"办公室租赁-XX大厦"）、起租日期、租赁年限等基本信息。

  2. 配置付款参数

    设置年付款额、付款频率（月/季/半年/年）、付款时点（期初/期末）。系统自动计算每期付款额。

  3. 设置折现率

    输入适用的折现率（通常为增量借款利率）。这是计算租赁负债现值的关键参数。

  4. （可选）添加其他参数

    如有复原成本、租赁激励或余值担保，勾选并填写对应金额。

  5. 自动计算

    系统自动计算：租赁负债现值、ROU 资产原值、以及完整的逐期摊销明细表。

  6. 查看摊销明细

    切换到「摊销明细」标签查看每期的利息费用、本金偿还、折旧金额和 ROU 净值。

  7. 批量管理

    可添加多个合同，每个独立计算。支持从 Excel 批量导入合同参数。

  8. 导出结果

    点击「导出 Excel」下载包含所有合同的完整测算工作簿（含摊销明细 + 会计分录参考）。

📐 计算原理租赁负债 = 各期付款按折现率折现到起租日的现值之和。ROU 资产 = 租赁负债 + 复原成本现值 - 租赁激励。折旧按直线法在租赁期内摊销。
💡 审计提示导出的 Excel 包含计算公式（非死数），可直接放入底稿归档。修改参数后公式自动更新。

6. 审计抽凭助手

MUS / 随机抽样双模式，单家 + 批量多公司，Seeded PRNG 可复现，导出日志与抽样清单

两种抽样方法

方法原理适用场景
🎯 MUS（货币单位抽样）按金额权重在累计数轴上投射样本，高价值项目自动高概率选中实质性测试、大额项目筛查
🎲 随机抽样Fisher-Yates 洗牌算法，给定种子可复现随机序列控制测试、设定规模抽样

参数说明

参数说明
风险系数 (RF)用于 MUS 方法计算样本量。可从内置映射表选择或手动输入
随机种子 (Seed)相同种子 + 相同数据 = 相同抽样结果。建议使用日期如 20260517
高价值阈值 (PM)金额超过此值的项目 100% 选取（MUS 模式）
样本量/比例设定规模随机抽样：直接指定样本数量；未设定规模：按 N=项目数×比例/RF 自动计算

操作步骤 — 单家公司

  1. 录入/导入数据

    手动逐行添加凭证（凭证号、金额、摘要），或从 Excel/CSV 批量导入。至少需要凭证号和金额两列。

  2. 选择抽样方法

    切换 MUS 或随机抽样标签。MUS 需设置 RF 和 PM；随机抽样需设置种子和样本量（或比例）。

  3. 执行抽样

    点击「执行抽样」。结果表格展示被选中的凭证列表，含命中原因（MUS 模式标注"高价值"或"MUS命中"）。

  4. 导出

    导出 Excel 包含「抽样清单」Sheet（被选中凭证明细）+「抽样日志」Sheet（参数、种子、命中原因）。

操作步骤 — 批量多家公司

  1. 上传批量文件

    上传包含多家公司数据的 Excel（每家公司一个 Sheet，或一列标识公司名称）。

  2. 配置参数表

    系统自动解析出所有公司，每个公司独立设置抽样参数（方法、RF、种子等）。支持向下填充快速配置。

  3. 批量执行

    点击「批量抽样」，逐公司执行并显示进度条。

  4. 导出

    导出 Excel 包含每个公司的抽样清单 + 抽样日志 + 抽样汇总 Sheet + 全局参数 Sheet。

💡 复现技巧需要复现上次抽样结果时，只需使用相同的随机种子和数据即可。建议将种子记录在审计底稿中。

7. TB 自动上数器

科目映射模板匹配、客户余额表自动填入 TB、公式保护、借贷自动校验

核心概念

科目映射模板：将客户自定义科目编码对应到标准 TB 科目的一套规则。维护一次，多期复用。

操作步骤

  1. 维护科目映射模板

    首次使用需建立映射表。手动添加科目映射（客户科目代码 → 标准科目代码），或从 Excel 导入已有映射。系统会保存到浏览器本地存储，下次打开仍在。

  2. 上传客户余额表

    上传客户的科目余额表 Excel/CSV。系统自动识别列头（科目代码、科目名称、期末余额、发生额等）。如有未识别列可手动指定。

  3. 上传 TB 底稿

    上传已有的 TB 底稿 Excel（或点「自动生成 TB 模板」创建一个空 TB 结构）。系统自动检测已有公式，公式单元格上数时会被跳过保护。

  4. 执行上数

    点击「自动上数」。系统根据映射模板匹配科目：资产/负债/权益类填入期末余额；收入/费用类填入本期发生额。自动处理借贷方向正负号。

  5. 查看未匹配项

    上数完成后，未匹配的科目以红色高亮展示，并提供模糊匹配 Top 3 推荐。可一键添加到映射模板并重新上数。

  6. 校验

    自动执行借贷平衡校验。如有不平衡会高亮提示差额。

  7. 下载

    导出填好的 TB（公式和格式完整保留）+ 未匹配清单 + 校验报告。

⚠ 关键注意TB 底稿中已有的公式单元格上数时会自动跳过，不会被覆盖。这是审计底稿数据完整性的核心保障。

8. BKD 底稿滚存与上数助手

BKD 底稿日期列自动识别、期末数→期初数滚存、科目映射上数、借贷校验

操作步骤

  1. 上传 BKD 底稿

    上传包含多个 Sheet 的 BKD 底稿 Excel。系统自动扫描表头，正则匹配日期列（支持 2024.12.31、2024-12-31、24.12.31 等格式）。

  2. 确认年份识别

    系统展示每个 Sheet 识别到的年份和日期列。如有识别错误可手动修正。

  3. 配置滚存规则

    指定源列（如"2024年末"）→ 目标列（如"2025年初"）。一个源列可对应多个目标列。

  4. 执行滚存

    系统将期末数硬拷贝为目标列期初数（死数），清空源列数据区（公式保留结构，死数清空），自动更新列标题至新年份。

  5. 导入本期余额表

    上传本期科目余额表，选择已维护的科目映射模板。系统自动匹配标准科目并在底稿中定位行。

  6. 执行上数

    根据科目映射自动填入本期数值。公式单元格自动跳过。根据借贷方向处理正负号。

  7. 校验

    自动校验借贷平衡和年初年末一致性。不平衡或钩稽不符的项目高亮报警。

  8. 下载

    导出完整的底稿 Excel（公式完整保留）+ 校验报告。

📋 滚存逻辑"滚存"是指将上期期末数复制为本期期初数。例如：2024年末审定数 → 2025年初数。这是 BKD 底稿年度更新的核心操作。

9. Word 报告上数与校验工作台

Word 附注表格自动上数、科目映射匹配、合计行加总校验、导出 .docx 和校验 Excel

操作步骤

  1. 上传 Word 报告

    上传 .docx 格式的审计报告。系统自动提取所有表格。也可直接粘贴 Word 表格内容。

  2. 配置列映射

    系统展示表格列表，为每个表格指定「科目名称列」（用来匹配）和「待填数值列」（目标填充位置）。

  3. 上传数据源

    上传审定 TB 或科目余额表 Excel 作为上数数据源。系统自动解析科目代码和金额列。

  4. 执行上数

    系统按科目名称自动匹配（去空格、全半角统一），将数据源数值填入报告表格空白单元格。已填单元格浅绿色标识，合计行浅黄色背景。

  5. 加计校验

    系统自动识别合计行，向上寻找明细行，逐列加总对比。差异超过 0.01 的单元格红底闪烁标记。

  6. 处理差异

    检查红底标记的单元格，判断是上数错误还是报告本身差异。可手动修正后重新校验。

  7. 导出

    导出填好的 Word 报告（.docx）+ 加计校验 Excel（校验报告 Sheet + 差异清单 Sheet）。

⚠ 注意系统按科目「名称」（而非代码）匹配，因为 Word 报告中通常不包含科目代码。匹配时已自动处理常见差异（空格、全半角括号等），但特殊写法可能需手动配对。

10. GDC 审计任务工时管理系统

四步闭环：唯一识别码 → 工时记录（自动带出类型/科目）→ 项目分摊 → 多维度管理仪表盘 + AI 智能问答

系统四步设计

  1. 第一步：建立项目与任务

    先在「项目信息管理」中添加项目（编号 + 名称）。再在「任务与工作流」中创建任务，选择关联项目、任务类型（DWW TOD/SAP/Sampling/Levvia），填写科目代码。点击「自动生成识别码」生成唯一编码（格式：项目编号_类型_科目_序号）。

  2. 第二步：成员填写工时

    切换到「工时记录」标签。选择任务后系统自动带出任务类型和负责科目（浅蓝底色只读），成员只需填写姓名、日期和工时。填报人输入过的名字会被自动补全。

  3. 第三步：配置项目分摊

    切换到「项目工时分摊」标签。一个任务Code 可分摊到多个项目，设置比例（总和应为 100%）。系统自动计算有效工时 = 原始工时 × 分摊比例。

  4. 第四步：查看仪表盘

    切换到「管理仪表盘」。4 个统计卡片 + 4 个图表（成员工时排名/类型分布/项目工时 TOP10/成员效能对比）+ 详细数据表。

11. 审计项目工时任务协同系统

多角色审计项目管理、任务分配、工时追踪、复核流程、一键导出快照归档

角色体系

角色权限与视角
所长/合伙人全局视图，所有项目和成员数据，关注整体效率
组长/经理管辖范围内项目，可分配任务、审批工时、查看组员效能
项目经理负责项目详情，创建任务清单、跟踪进度
复核人审核已完成任务，确认或退回
审计员查看分配给自己的任务，上报进度、填写工时

操作步骤

  1. 切换角色

    顶部角色选择器切换身份。不同角色看到不同的数据和操作界面。

  2. 创建项目

    （项目经理）填写项目信息，可勾选「从科目映射自动生成任务清单」快速创建标准化任务列表。

  3. 分配任务

    （组长/经理）将任务分配给团队成员。支持均分按钮一键平均分配。每个任务可指定负责人和截止日期。

  4. 报工与追踪

    （审计员）在自己的「今日待办」中看到任务，完成后上报进度和工时。系统自动累加工时和更新任务状态。

  5. 复核闭环

    （复核人）审核已完成任务，确认通过或退回修改。通过后任务正式关闭。

  6. 导出归档

    （组长/经理）一键导出全景快照 Excel（含项目概览 + 任务进度 + 工时汇总 + 问题日志 4 个 Sheet）+ JSON 备份 + PDF 报告。归档前自动完整性校验。

📦 数据存储所有数据存储在浏览器 localStorage。不同角色在同一浏览器中切换即生效。数据可通过 JSON 备份跨设备迁移。

12. 智能待办

AI 自然语言解析任务、桌面提醒通知、邮件格式智能识别、CSV/JSON 导入导出

功能一览

功能说明
🤖 AI 解析粘贴自然语言或邮件，自动提取任务、截止日期、优先级、标签
📧 邮件识别自动跳过 From/To/Subject 等邮件头、签名、免责声明，只解析正文中的任务
🔔 桌面提醒浏览器通知权限授权后，定时弹出待办提醒（可配间隔和时段）
📥 批量导入CSV 格式粘贴、JSON 文件导入、示例数据加载
📤 导出备份导出 CSV / JSON，可用于数据迁移
🎯 优先级高/中/低三级，逾期红色标记，临近黄色提醒

操作步骤 — AI 解析

  1. 粘贴文本

    在顶部文本框粘贴自然语言（如"明天下午3点前完成审计报告，高优先级 #审计"）或整封工作邮件。

  2. 点击 AI 解析

    系统优先尝试本地 AI 模型。如模型未就绪则自动降级为本地正则解析（同样有效）。

  3. 查看结果

    解析出的任务自动添加到待办列表，包含任务名称、截止日期、优先级和标签。

操作步骤 — 设置桌面提醒

  1. 打开设置

    点击右上角齿轮图标 ⚙️。

  2. 请求权限

    点击「请求权限」→ 浏览器弹出通知权限提示 → 点击「允许」。

  3. 测试提醒

    点击「🧪 测试提醒」验证通知是否正常弹出（屏幕右下角）。

  4. 配置参数

    设置每日首次提醒时间（如 9 点）、提醒间隔（如 4 小时）、提前提醒截止前小时数（如 24 小时）。

  5. 保持页面打开

    页面需保持打开状态（可最小化），系统会按设定的间隔自动检查并推送提醒。
